Developers Track
================

If you're extending exo2micro, scripting batch runs, tuning
parameters for a new sample type, or writing analysis code on
top of the pipeline, this is your track. It covers the conceptual
model, the scripting API, the full parameter reference, how the
scale methods work internally, and autodoc-generated module
references.
