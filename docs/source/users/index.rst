Users Track
===========

If you're running microscopy samples and want to go from raw TIFFs to
difference images without writing Python, this is your track. It covers
installation, your first run using the GUI, how to read the diagnostic
plots, how to zoom in on regions of interest, and what to do when
things don't look right.
